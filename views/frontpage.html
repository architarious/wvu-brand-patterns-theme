WVU Brand Patterns

WVU Brand Patterns is a theme for making WVU-branded websites. It is built with Sass and works with WVU’s CleanSlate content management system.

GitHub Examples

Quickly develop website themes using basic styles, utilities, and HTML partials.

Patterns use Radius tags. They look like this:

  <r:partial name="patterns/wvu-masthead" />

Which outputs the following:

  <div class="wvu-masthead">
    <div class="wvu-masthead__container">
      <a class="wvu-masthead__logo" href="http://www.wvu.edu/"><span class="wvu-masthead__university-name">West Virginia University</span>Digital Services</span></a>  
      <r:partial name="patterns/wvu-masthead__search-bar" />
    </div>
  </div>

And is styled by referencing a Sass partial:

  <r:partial name="scss/2-modules/wvu-masthead" />

The WVU Brand Patterns theme includes patterns for the official WVU masthead, hero unit, sub features, navigation, typography, WVU footer, and various other elements, which can be viewed here, along with the required code.

Instructions:

  1. Clone/download the WVU Brand Patterns theme from GitHub or Stash.
  2. Create a repo for your new in the CleanSlate Themes project on Stash.
  3. Copy the files you want from the WVU Brand Patterns theme into your new theme.
  4. Edit your theme and compile your changes using your favorite Sass compiler.
  5. Test your theme in Hammer.
  6. Push and sync your theme.

The WVU Brand Patterns theme comes with Boubon/Neat by default. Want to use your own grid instead? No problem, just remove the references to Bourbon and Neat from the plugins.sass and layouts-dir.sass file and include your own grid.
